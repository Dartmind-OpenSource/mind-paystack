### Payment Channels

Discover supported channels and run direct channel charges (card, bank, USSD, QR, mobile money, bank transfer).

#### List available channels

```dart
final channels = await MindPaystack.instance.paymentChannel.listPaymentChannels(
  country: 'NG',
  currency: 'NGN',
  amount: 200000,
);
```

#### Banks and Mobile Money

```dart
final banks = await MindPaystack.instance.paymentChannel.listBanks(currency: 'NGN');
final momo = await MindPaystack.instance.paymentChannel.listMobileMoneyProviders(country: 'GH');
```

#### Card charge

```dart
final charge = await MindPaystack.instance.paymentChannel.chargeCard(
  amount: 200000,
  email: 'user@example.com',
  cardNumber: '4084084084084081',
  cvv: '408',
  expiryMonth: '12',
  expiryYear: '30',
  pin: '1234',
);
```

#### USSD, QR, Mobile Money, Bank Transfer

```dart
await MindPaystack.instance.paymentChannel.chargeUssd(amount: 200000, email: 'user@example.com', bankCode: '058');
await MindPaystack.instance.paymentChannel.chargeQr(amount: 200000, email: 'user@example.com', provider: 'visa');
await MindPaystack.instance.paymentChannel.chargeMobileMoney(amount: 200000, email: 'user@example.com', provider: 'mtn', phoneNumber: '2335...');
await MindPaystack.instance.paymentChannel.chargeBankTransfer(amount: 200000, email: 'user@example.com');
```

#### Submit and validate charge

```dart
await MindPaystack.instance.paymentChannel.submitChannelCharge(
  reference: charge.reference,
  type: 'otp',
  value: '123456',
);

await MindPaystack.instance.paymentChannel.validateChannelCharge(
  reference: charge.reference,
  validation: {'avs_address': '...', 'avs_zip': '...'},
);
```
