### Payment Methods

Store and manage reusable payment methods for customers.

#### List methods

```dart
final list = await MindPaystack.instance.paymentMethods.listPaymentMethods(
  customerId: 'cust_123',
  page: 1,
  perPage: 20,
);
```

#### Create from token

```dart
final method = await MindPaystack.instance.paymentMethods.createPaymentMethod(
  customerId: 'cust_123',
  type: 'card',
  token: 'pm_token_xxx',
  metadata: {'label': 'Primary card'},
);
```

#### Update

```dart
final updated = await MindPaystack.instance.paymentMethods.updatePaymentMethod(
  id: method.id,
  updates: {'label': 'Work card'},
);
```

#### Remove

```dart
await MindPaystack.instance.paymentMethods.removePaymentMethod(method.id);
```
