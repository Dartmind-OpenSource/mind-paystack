### Entities & Models

Strongly typed models across features (subset shown):

- Transactions: `Transaction`, `TransactionResponse`, `TransactionEvent`, `TransactionSession`, `TransactionTotals`, `AuthorizationResponse`, `RefundResponse`
- Charges: `ChargeResponse`, `ChargeStatus`
- Payment Channels: `PaymentChannel`, `Bank`, `MobileMoney`, `CardBinInfo`, `ChannelFees`, `AccountResolution`, `BinResolution`
- Payment Methods: `PaymentMethod`
- Errors: `MindException`, `ValidationError`, `ErrorMetadata`

Refer to package sources under `packages/mind_paystack/lib/src/features/**/models` for fields.
