# MindPaystack

MindPaystack is a modern Dart SDK for Paystack, designed with clean architecture, dependency injection, and extensibility in mind.

It powers:
- **mind_paystack** → Core Dart SDK
- **mind_paystack_flutter** → Flutter integration
- Future variants (e.g. web, server-side)

## Features
- Strongly typed API responses
- Dependency injection with `injectable`
- Modular service architecture
- Unified error handling via `MindException`
- Works across Dart and Flutter projects
