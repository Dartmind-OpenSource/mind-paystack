### Flutter Quickstart

1) Add dependency:

```sh
dart pub add mind_paystack_flutter
```

2) Initialize in `main`:

```dart
await MindPaystack.init(
  PaystackConfig(publicKey: 'pk_test_xxx', secretKey: 'sk_test_xxx'),
);
```

3) Create a payment:

```dart
final tx = await MindPaystack.instance.transaction.initializeTransaction(
  amount: 500000,
  email: 'user@example.com',
  currency: 'NGN',
);
```

4) Open `tx.authorizationUrl` and then verify with `verifyTransaction`.
