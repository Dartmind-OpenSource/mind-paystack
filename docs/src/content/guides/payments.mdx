### Payments

End‑to‑end flows for collecting payments with Paystack using the SDK.

#### Card payments (Charge API)

Use the channel‑agnostic Transaction flow for hosted checkout, or use Charge when you manage the steps yourself.

Initialize via hosted checkout:

```dart
final tx = await MindPaystack.instance.transaction.initializeTransaction(
  amount: 150000,
  email: 'user@example.com',
  currency: 'NGN',
);
// Open tx.authorizationUrl
```

Charge directly (advanced):

```dart
final charge = await MindPaystack.instance.paymentChannel.chargeCard(
  amount: 150000,
  email: 'user@example.com',
  cardNumber: '4084084084084081',
  cvv: '408',
  expiryMonth: '12',
  expiryYear: '30',
  pin: '1234',
);

// For OTP/PIN/address validation flows, follow up with:
// submitChannelCharge(reference: charge.reference, type: 'otp'|'phone'|'birthday', value: '...')
// validateChannelCharge(reference: charge.reference, validation: {...})
```

#### Verify payments

```dart
final result = await MindPaystack.instance.transaction.verifyTransaction(tx.reference);
```

#### Partial debit

```dart
final res = await MindPaystack.instance.charge.createPartialDebit(
  amount: 50000,
  email: 'user@example.com',
  authorizationCode: 'AUTH_XXXX',
  currency: 'NGN',
);
```
